# Cartoonify Me 🎨

Turn real photos into stylized cartoon characters and generate printable invitations.

## V1 Focus
Birthday Invitation Generator

- Upload photo
- Cartoon transformation (WIP)
- Cinderella-themed invitation
- Export as image (PNG)
