# Cartoonify Me 🎨

Cartoonify Me is a web application that transforms user photos into stylized cartoon portraits and generates themed birthday invitations.

## Overview

This project explores how AI-generated content can be integrated into a simple, user-facing workflow — from image upload to final downloadable output.

It focuses on building a lightweight end-to-end system that combines:
- user interaction
- AI image generation
- dynamic content rendering

## Tech Stack

- Node.js (backend API)
- OpenAI Image API (image generation pipeline)
- HTML/CSS/JavaScript (frontend UI)
- html2canvas (image rendering & download)

## Key Features

- Multiple themed outputs (Princess, Candyland, Superhero, etc.)
- Image upload and transformation pipeline
- Dynamic invitation generation
- Downloadable final output

## What I Focused On

- Designing a simple pipeline to handle input → transformation → output
- Managing variability in AI-generated results and making outputs usable
- Building a complete user flow from upload to download
- Iterating quickly on UI and output quality

## Notes

This project reflects my interest in building small, end-to-end systems and exploring how AI can be integrated into practical user experiences.
